# Marwahs Odoo26

Logistics / fleet management project for the Odoo26 hackathon.

## Structure

- `backend/` — Node.js / Express API (auth, vehicles, drivers, trips, dashboard)
- `frontend/` — client-side application (to be added)

## Getting started (backend)

```bash
cd backend
cp .env.example .env   # then fill in your values
npm install
npm run dev
```

## Team

Two-person team: backend and frontend work is split across the two folders above.
